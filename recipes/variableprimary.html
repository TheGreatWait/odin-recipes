Variable Primary Systems

Courtesy of jmpcoblog.com

System Description

It is recommended that you learn about the Primary Secondary system first before reading about this one. They share many of the same characteritics.

From an equipment point of view, the Variable Primary system is the simplest of the three that we will look at today. It only consists of one main loop. The primary pumps provide a variable flow of water to the chillers. The chiller control valves modulate to control exactly how much water goes to which. While this is changing in the industry, most chillers do not support variable flow, so each control valve will either be completely open or completely closed.

Cool water from the chillers now has two options: it can either keep flowing to the users, or it can bypass the users and head directly back to the primary pumps to be recirculated. The amount of water going each way is controlled by the bypass control valve, which is arguably the most critical part of this system. If it does not work correctly, the system will never be able to probide any meaningful cooling to users. It will either over-cool (using too much power) or under-cool, decreasing overall system performance.

This delicate balance between 1) the number and speed of the primary pumps, 2) the number of chillers operating, 3) whether the chiller control valves are opened and closed, and 4) the bypass valve status, makes the variable primary the easy to learn / difficult to master system in the HVAC world.

Advantages

(In no particular order)

  • This system is most efficient cooling hydronic setup possible without spending an arm and a leg on exotic chiller setups
  • Because this system has so little equipment, it is relatively cheap to install

Disadvantages

(Ordered by most critical to least)

  1. Variable Primary systems are simple from a layout view, but are operationally complex. Everything needs to work right for the system to function correctly.
  2. The system can be more expensive to expand than a Primary Secondary System. Booster pumps may be required for far-out users, which negates the benefits of this system in the first place.
  3. Control and equipment problems (which will develop as the system ages) can have a stark effect on system efficiency.
  4. Not all chillers can support variable flow, which constrains system operating flexibility.
